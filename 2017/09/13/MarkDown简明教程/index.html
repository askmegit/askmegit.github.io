MarkDown简明教程

发表于 2017-09-13

本文主要用于记录MarkDown语法,在使用Cmd Markdown编辑器时,同事学习了一下其高级语法.
在使用#,*,+,-这些标记符号时,有时与文本保留一个空格是很有必要的.

1. 标题

# 一级标题

一级标题

###### 六级标题

六级标题


 如果标题没有那么多的情况下可以使用==或者--表示一级和二级标题

3. 图片与连接

插入链接与插入图片的语法很像，区别在一个 !号

插入图片:

插入内链:

插入外链:
[askme的博客][1]
注意这里有空格,否则会影响后面md标记语法
[1]: http://askme.xin

直接使用链接: 
`<http://askme.xin>`

16. 甘特图

甘特图内在思想简单.基本是一条线条图, 横轴表示时间,纵轴表示活动(项目),线条表示在整个期间上计划和实际的活动完情况.它直观地表明任务计划在上面时候进行,及实际进展与计划要求的对比.
